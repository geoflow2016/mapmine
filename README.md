# 全球矿业项目开源地图V1.0

基于 ICMM Global Mining Dataset (September 2025) 制作的全球矿山分布可视化网站。

## 数据来源

- **数据集**: ICMM Global Mining Dataset V1
- **记录数**: 8,508 条矿点数据
- **覆盖范围**: 129 个国家/地区，50+ 种矿产商品
- **属性**: 矿山名称、地理位置、所属集团、商品类型、置信度等级等

## 在线访问

部署完成后访问: `https://你的用户名.gitee.io/全球矿业项目开源地图`

## 部署步骤 (Gitee Pages)

### 方式一：网页上传（推荐）

1. 访问 [Gitee](https://gitee.com) 并登录
2. 点击右上角 **"+"** → **"新建仓库"**
3. 填写仓库信息：
   - 仓库名称：`global-mining-map`
   - 仓库路径：`https://gitee.com/你的用户名/global-mining-map`
   - 私有/公开：选择"公开"
   - 勾选"使用 README 初始化仓库"
4. 点击"创建"
5. 进入仓库后，点击"上传"按钮
6. 将本仓库中的 `index.html` 文件拖入上传区域
7. 点击"提交"
8. 进入 **"服务"** → **"Gitee Pages"**
9. 点击"启动"
10. 访问 `https://你的用户名.gitee.io/global-mining-map`

### 方式二：Git 克隆

```bash
# 1. 克隆仓库
git clone https://gitee.com/你的用户名/global-mining-map.git
cd global-mining-map

# 2. 复制 index.html 到仓库目录
cp ../index.html .

# 3. 提交推送
git add .
git commit -m "添加全球矿业地图"
git push

# 4. 在 Gitee Pages 设置中启用
```

## 功能特性

- 🗺️ **交互式地图** - ESRI卫星图/街道图切换
- 🔍 **多维筛选** - 国家/地区、商品类型、资产类型、置信度
- 📊 **统计图表** - 商品分布、国家分布、置信度分布
- 📋 **矿点列表** - 分页浏览，快速定位
- 🌐 **中文界面** - 全中文数据展示

## 数据字段说明

| 字段 | 说明 |
|------|------|
| ICMMID | ICMM全球唯一标识 |
| 矿山名称 | 矿山/冶炼厂/精炼厂名称 |
| 集团名称 | 所属矿业集团 |
| 纬度/经度 | 地理坐标 |
| 资产类型 | 矿山/冶炼厂/精炼厂等 |
| 主要商品 | 第一产出商品 |
| 次要商品 | 第二产出商品 |
| 置信度等级 | 高/中/很低 |

## 技术栈

- Leaflet.js - 交互式地图
- Chart.js - 统计图表
- ESRI 地图服务 - 卫星/街道图层
- 纯前端，无需后端

## 注意事项

- 数据来源于ICMM公开数据集，可能存在部分坐标缺失
- 置信度等级说明数据精确程度
- 仅供研究参考，实际决策请参考官方数据
